# 📘 코딩테스트 준비 커리큘럼 (2025 최신)

코딩테스트는 **문제 유형을 어떤 순서로 공부하느냐**가 실력 향상 속도를 결정합니다.  
아래 커리큘럼은 초보 → 중급 → 실전 단계까지 가장 효율적인 학습 순서로 구성되어 있습니다.

---

# 1️⃣ 기본기 다지기 (1~2주)

> **목표:** 문자열/반복문/조건문/배열/구현 등 ‘코딩의 기초 체력’ 완성  
> **난이도:** 브론즈 상위 ~ 실버 5

### ✔ 학습 범위

- 입출력 숙련
- 문자열 처리
- 기본 수학
- 정렬 기초
- 배열/리스트 기초
- 브루트포스 맛보기

### ✔ 추천 문제 유형

- 문자열 조작
- 배열 처리
- 단순 구현
- 기본 정렬
- 브루트포스 기초

---

# 2️⃣ 핵심 알고리즘 입문 (3~4주)

> **목표:** 기본 자료구조 + 탐색 알고리즘 이해  
> **난이도:** 실버 5 ~ 실버 3

### ✔ 학습 범위

- 스택 / 큐 / 덱
- DFS / BFS
- 정렬 심화 (커스텀 정렬)
- 재귀 기초

### ✔ 추천 문제 유형

- 스택/큐 문제
- DFS/BFS 탐색 문제
- 재귀/백트래킹 맛보기
- 정렬 응용 문제

---

# 3️⃣ 대표 패턴 학습 (핵심 구간)

> **목표:** 문제를 보면 알고리즘 방향이 떠오르는 ‘코딩테스트 감각’ 만들기  
> **난이도:** 실버 3 ~ 실버 1

### ✔ 학습 범위

- 백트래킹(순열/조합)
- 그리디
- 이분 탐색
- 누적합
- 슬라이딩 윈도우
- 투 포인터
- DP 입문

### ✔ 추천 순서

1. 백트래킹
2. 그리디
3. 이분 탐색
4. 누적합 / 슬라이딩 윈도우
5. DP 기초

---

# 4️⃣ 중급 알고리즘 (골드 구간)

> **목표:** 실전 기업 코테의 대부분 문제 대응  
> **난이도:** 골드 5 ~ 골드 3

### ✔ 학습 범위

- 다익스트라
- 플로이드 워셜
- 최소 스패닝 트리 (크루스칼/프림)
- 위상 정렬
- DP 심화
- 투 포인터 고급

### ✔ 필수 알고리즘

- 그래프 최단거리
- MST
- 위상 정렬
- DP(배낭/구간/누적 DP)

---

# 5️⃣ 실전 기출 학습 (최종 단계)

> **목표:** 카카오, 네이버, 라인, 삼성 A형 수준 문제 해결  
> **난이도:** 골드 3 ~ 플레 5

### ✔ 학습 범위

- 카카오 기출 (2018~2024)
- 프로그래머스 Lv3
- 삼성 SW 역량테스트 A형 기출
- 백준 골드 상위 문제

### ✔ 학습 전략

- 유형별 기출 패턴 분석
- 코드 개선(Refactoring)
- 시간복잡도 기준으로 설계하는 연습
- 실전처럼 제한 시간 두고 풀이

---

# 📅 4개월 완성 로드맵

## 📌 1개월차 — 기본기 + DFS/BFS

- 문자열/배열/구현 숙련
- 스택/큐/덱
- DFS/BFS
- 브루트포스/백트래킹 입문

## 📌 2개월차 — 그리디 + 이분탐색 + 슬라이딩 윈도우

- 그리디 패턴 정복
- 이분 탐색/파라메트릭 서치
- 누적합/투 포인터/슬라이딩 윈도우
- DP 기초 문제 풀이

## 📌 3개월차 — 그래프 알고리즘 + DP 심화

- 다익스트라
- 플로이드-워셜
- MST
- 위상 정렬
- DP 심화(배낭/구간)

## 📌 4개월차 — 기출 실전 대비

- 카카오 기출 집중
- 삼성 A형 3개년
- 프로그래머스 Lv3
- 백준 골드3~골드1 문제 집중
